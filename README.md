# ArchRice

My dotfiles for ArchLinux ricing with the Hyprland WM

![Rice Showcase](https://cloud-g3lb4efvp-hack-club-bot.vercel.app/0terminals.png)

### Dependencies

- hyprland
- hyprpaper
- kitty
- starship
- waybar
- ttf-cascadia-code-nerd
- woff2-font-awesome
- firefox
- nemo
- oxygen-cursors
- wofi
- udiskie
- udisks2
- swaync
- xorg-xrdb
- brightnessctl
- networkmanager

- cava
- nano
- pavucontrol
- cliphist
- fastfetch
